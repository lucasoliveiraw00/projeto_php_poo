# Projeto PHP POO

## Características

* CRUD de Pessoas

## Tecnologias Utilizadas

* PHP orientado a objetos
* Arquitetura MVC
* URL amigável
* htaccess
* Banco de dados MYSQL
* Linguagem de consulta SQL
* Bootstrap
* JQuery

## Uso

* Importar banco de dados
* Usar um servidor web Apache e MYSQL exemplo xampp
